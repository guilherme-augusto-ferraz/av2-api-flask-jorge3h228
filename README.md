# av2-api-flask

4. Estrutura do Projeto
O projeto segue uma estrutura modular com boa organização de código.

Sugestão de estrutura criada:

```
app.py
database.py
models/
	├── __init__.py
	├── user.py
	└── registro.py
routes/
	├── __init__.py
	├── users.py
	└── registros.py
requirements.txt
```

Instruções rápidas:

- Instale dependências: `pip install -r requirements.txt`
- Rode a API: `python app.py`
